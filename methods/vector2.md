# Vector2

**Vector2** used to describe point on screen. Has two float's\(x and y axis\)

## Vector2.new\(float x, float y\):

```text
        Creates new Vector2 object (x, y)
```

## float x

```text
        x value of object
```

## float y

```text
        y value of object
```

## float Length\(\)

```text
        Returns Length ( sqrt(x*x + y*y) )
```
