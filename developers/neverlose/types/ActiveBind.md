# ActiveBind

{% hint style="info" %}
You can access `ActiveBind` instance through `GetBinds` [function](../Cheat.md)
{% endhint %}

## Functions

## GetName

### Return value:

| Name | Type | Description |
| :--- | :--- | :--- |
| name | string | Bind Name |

```lua
bind:GetName()
```

## GetValue

### Return value:

| Name | Type | Description |
| :--- | :--- | :--- |
| value | string | Bind value |

```lua
bind:GetValue()
```

## IsActive

### Return value:

| Name | Type | Description |
| :--- | :--- | :--- |
| value | bool | Is keybind active |

```lua
bind:IsActive()
```
